# 🎉 Shanzay's Birthday Countdown

A minimalist birthday countdown website created with love for my best friend Shanzay!

## Features

✨ **Countdown Timer** - Live countdown to the special day (August 11, 2026)
📸 **Photo Gallery** - Share memories with placeholder photos
💌 **Fun Birthday Message** - Personalized greeting and celebration vibes
🎨 **Minimalist Design** - Clean, modern, and mobile-responsive

## How to Use

1. Visit the website via GitHub Pages
2. Watch the countdown to Shanzay's birthday
3. Click on any photo placeholder to add your favorite memories
4. Share and celebrate! 🎂

## Setup

This site is hosted on GitHub Pages. Just push updates to the repository and they'll go live automatically!

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
